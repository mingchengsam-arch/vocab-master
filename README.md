# Vocabulary Learning Application

## Instructions on how to use the vocab learning application

1. **Installation**  
   - Clone the repository: `git clone https://github.com/mingchengsam-arch/vocab-master.git`  
   - Navigate into the project directory: `cd vocab-master`  
   - Install dependencies: `npm install` (or use `pip install` for Python)

2. **Running the Application**  
   - For JavaScript: `npm start`  
   - For Python: `python main.py`

3. **Using the Application**  
   - Follow the on-screen instructions to add vocabulary words.
   - You can test your vocabulary knowledge through quizzes.
   
4. **Contributing**  
   - If you would like to contribute, feel free to fork the repository and submit a pull request.
   
5. **Contact**  
   - For further questions, reach out via GitHub Issues or contact the repository owner directly.

Happy Learning!
